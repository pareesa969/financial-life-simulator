# 💰 Financial Life Simulator

An interactive web-based financial decision game built using **HTML, CSS, and JavaScript**.

This simulator allows users to experience real-life financial choices and understand how decisions impact their **savings, debt, and investments** over time.

---

## 🚀 Overview

The Financial Life Simulator is a lightweight, browser-based game designed for quick gameplay (1–2 minutes).
Players go through multiple financial scenarios and make decisions that shape their financial future.

---

## 🎯 Features

* 🎮 Interactive financial decision gameplay
* 🔀 60 realistic financial scenarios
* 📍 10 scenarios per game session
* 📊 Real-time financial dashboard
* 📈 Dynamic line chart using Chart.js
* 🧠 Score calculation system
* 🏆 Player classification system:

  * The Saver
  * The Investor
  * The Risk Taker
  * The Spender
* 📌 Progress tracking (Scenario 1 → 10)
* 🔁 Replay functionality
* 🎨 Modern UI with animations and hover effects

---

## 🧠 Game Logic

Each decision impacts the player's:

* 💰 Savings
* 💳 Debt
* 📈 Investments

### Score Formula

Score = Savings + Investments - Debt

---

## 🏁 Game Flow

1. Start the game
2. Enter your name and starting money
3. Play through **10 randomly selected scenarios**
4. Make financial decisions
5. Track financial changes via dashboard and chart
6. View final score and financial identity
7. Replay the game

---

## 📊 Financial Visualization

* Real-time chart updates after each decision
* Final chart displayed on result screen
* Tracks:

  * Savings
  * Debt
  * Investments
