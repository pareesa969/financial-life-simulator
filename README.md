# Financial Life Simulator

A web-based financial decision game built with Python and Streamlit.

## Features

- Interactive financial decision game
- Random scenario system
- Financial dashboard
- Score evaluation system
- Replay functionality
- Hackathon friendly

## Installation

1. Clone repo

2. Create environment

python -m venv venv

3. Activate environment

source venv/bin/activate

4. Install dependencies

pip install -r requirements.txt

## Run

streamlit run app.py

## Game Flow

1. Start Game
2. Enter name and starting money
3. Play 4 financial scenarios
4. Make decisions
5. View final financial score
6. Replay
